# üß† AI Quiz Generator

A full-stack **AI-powered Quiz Platform** that scrapes Wikipedia articles and generates multi-level quizzes using **Google Gemini API**.  
Built with **FastAPI (backend)**, **React + TailwindCSS (frontend)**, and **MySQL**, fully containerized with **Docker & Docker Compose**.

---

## üöÄ Features
‚úÖ Scrapes and structures Wikipedia content  
‚úÖ Generates quizzes using Gemini via LangChain  
‚úÖ Semantic search with FAISS  
‚úÖ Stores and retrieves quizzes from MySQL  
‚úÖ Beautiful frontend UI built with React + TailwindCSS  
‚úÖ Complete Dockerized setup for local and cloud deployment

## üß≠ Application Flow

The **AI Quiz Generator** follows a modular architecture connecting the **Frontend**, **Backend**, **Database**, and **Gemini AI model**, enabling smooth interaction between data scraping, quiz generation, and visualization.

```mermaid
flowchart TD
    %% ====== MAIN USER FLOW ======
    Z[üìä Dashboard] --> A[üßë‚Äçüíª User Enters Wikipedia URL]
    A --> B[üîç Data Scraped from Wikipedia and Stored in MySQL Database]
    B --> W[üíæ MySQL Database]
    B --> C[üéöÔ∏è User Selects Difficulty Level & Sections]
    C --> D[üì§ Backend Sends Request to Gemini API for Quiz Generation]
    D --> Y[ü§ñ Gemini AI Model via LangChain]
    D --> E[‚è≥ Displays Summary While Waiting for AI Response]
    Y --> F[üß† Quiz JSON Generated by Gemini]
    F --> G[üóÑÔ∏è Store Generated Quiz in MySQL Database]
    G --> H[üì¶ Backend Returns Structured Quiz Response]
    G --> W
    H --> I[ü™Ñ Frontend Displays Summary & Questions]
    E --> I
    I --> J[üìù User Takes Test and Views Score]

    %% ====== HISTORY FLOW ======
    Z --> K[üìú History Section]
    K --> L[üìÇ Display All Generated Quizzes in Table Format]
    W --> L
    L --> M[üîé View Quiz Details]
    M --> I
```

## ‚ùì Troubleshooting
Issue	Cause	Fix

CORS error	API origin mismatch	Allow CORS in FastAPI and update VITE_API_BASE_URL
MySQL connection error	Wrong host in DATABASE_URL	Use db (not localhost) in the connection string
